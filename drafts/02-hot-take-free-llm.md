I tested 16 free LLM providers so you don't have to.

Google's Gemini is fast but inconsistent.
Groq is blazing fast but limited in context.
Cerebras handles code better than anything else.
Mistral wins at reasoning on small models.

But none of them are reliable alone.

So I built something stupid simple:
One API endpoint that aggregates all 16 providers.
OpenAI-compatible. Drop-in replacement.
Automatic failover when one goes down.

Cost? $0. All free tiers.
